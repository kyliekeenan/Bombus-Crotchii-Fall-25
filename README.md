# Bombus-Crotchii-Fall-25
Phenology of Bombus Crotchii California Spatial Analysis
This is a project to use data on the endangered _Bombus crotchii_ in Central California with a focus on Santa Barbara County.
It was started by Kylie Keenan and Katja Seltmann in Summer 2025 and joined by Gretchen LeBuhn in Fall 2025.

**Milestones:**
  Gathered data from iNaturalist

Downloaded specimen only data from GBIF on October 13, 2025
